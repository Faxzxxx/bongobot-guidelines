BongoBot Privacy Policy

Last updated: April 11, 2025

1. Information We Collect

BongoBot collects minimal information necessary to provide functionality. This includes:

  • User IDs and usernames for moderation logs and case tracking
  • Message content only if your server uses features like bad word filtering or custom moderation logs
  • Guild (server) ID and channel IDs for configuration storage

2. How We Use the Data

The collected data is used solely for the following purposes:

  • Tracking moderation cases (warns, bans, etc.)
  • Providing features like automatic muting or word filters
  • Displaying audit logs and status of reports

3. Data Sharing

BongoBot does not share any data with third parties. All data is stored locally and is not used for analytics or advertising.

4. Data Deletion

If you wish to remove your data from BongoBot, you can:

  • Remove the bot from your server (this clears configuration)
  • Contact the developer for manual deletion by emailing: bongobot@support.dev

5. Contact

If you have any questions regarding this policy, please contact us via our Support Server.
